## О проекте
Проект создан с использованием Pug-шаблонизатора и SCSS-препроцессора. Сборка осуществляется с помощью Gulp.

**Стек**:  
HTML, SCSS, Pug, Gulp


### Установка и запуск

```
npm install
npm run dev
```
